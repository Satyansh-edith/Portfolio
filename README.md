# 💻 Satyansh Dubey - Personal Portfolio

This is the repository for my developer portfolio, presenting a collection of high-performance full-stack applications, interactive user experiences, and AI integrations.

## 🛠️ Technology Stack
* **Framework**: Next.js 16 (App Router)
* **Frontend**: React 19, TypeScript, Tailwind CSS v4, Framer Motion
* **Graphics**: Three.js & Custom WebGL Shaders (e.g. Warp Drive, Particle Text Effects)

## ⚙️ Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the development server.

### 3. Production Build
```bash
npm run build
```
